## zh-CN

使用 `MonthPicker` 实现月选择器。

## en-US

You can get a month selector by using `MonthPicker`.

````jsx
import { DatePicker } from 'antd';
const MonthPicker = DatePicker.MonthPicker;
ReactDOM.render(
  <MonthPicker defaultValue="2015-12" />
, mountNode);
````
